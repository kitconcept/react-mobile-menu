# react-mobile-menu

Mobile Menu for React.

Dependencies:

```
npm install hamburgers --save
npm install react-reveal --save
```

Usage:

```JavaScript:
import ReactMobileMenu from "./../ReactMobileMenu";

const MyReactComponent = () => (
  ...
  <ReactMobileMenu />
  ...
);
```
